== Description ==

A minimal and responsive HTML5 landing page built on lightweight, clean and customizable code.

== Folders ==

* /build: This is the compiled version. Includes regular CSS and ready to be uploaded to your server.
* /source: Use this folder if you're used to work with Sass.

== Changelog ==

= 1.3.0 - 4/3/2018 =
* Fixed role markup for HTML validation.
* Added a class to add shadow to an element: shadow-s, shadow-m and shadow-l.

= 1.2.0 - 4/3/2018 =
* Cards now have a .card-content class which handles the padding. Makes it easy to add images without padding.
* Updated Sketch file.

= 1.1.0 - 21/2/2018 =
* Improved accessibility by adding role attributes.
* Now easy to use a logo as text, image or svg.
